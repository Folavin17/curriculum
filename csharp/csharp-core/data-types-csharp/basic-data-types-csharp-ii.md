# int & double

---

## Content


Integers are whole numbers such as `1`, `2`, `-30`, `9999`:
```cpp
int number = 10;
Console.WriteLine(number);
// 10
```

### double
---

The `double` type represents floating-point numbers, which is any number that contains a decimal point.
```cpp
double number = 10.5;
Console.WriteLine(number);
// 10.5
```
