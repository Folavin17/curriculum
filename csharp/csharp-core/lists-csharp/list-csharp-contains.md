# List Methods

---

## Content

We can check if a list contains a certain element with `.Contains()`:

```csharp
List<int> numbers = new List<int>();
numbers.Add(6);

Console.WriteLine(numbers.Contains(6))
// Output: True


Console.WriteLine(numbers.Contains(12))
// Output: False
```
