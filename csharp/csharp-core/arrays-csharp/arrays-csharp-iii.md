# Arrays

---

## Content

We can also reassign values of an array directly:
```csharp
int[] numbers = new int[] {1, 2, 3, 4, 5};
Console.WriteLine(numbers[0]);  
// Output: 1
numbers[0] = 25
Console.WriteLine(numbers[0]);  
// Output: 25
```
