# Let's Create Our First Program!

---

## Content

Ready for some code?

We'll write out code in the `Program.cs` file.

In **C#**, you can print text to the screen using the `Console.WriteLine` method:

```csharp
using System;

class Program
{
  static void Main()
  {
    Console.WriteLine("Hello, World!");
  }
}
// Hello, World!
```
