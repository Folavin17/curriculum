# as.double

---

## Content

The `as.double` converts an object to double (floating point) type.
```r
# Convert a number to a double
x <- 123
x <- as.double(x)
print(x)  
# 123.0

# Convert a vector of numbers to a vector of doubles
v <- c(1, 2, 3)
v <- as.double(v)
print(v)  
# 1.0 2.0 3.0
```
