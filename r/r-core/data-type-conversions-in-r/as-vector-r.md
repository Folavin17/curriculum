# as.vector

---

## Content

The `as.vector` converts an object to vector type.

```r
# Convert a number to a vector
x <- 123
x <- as.vector(x)
print(x)  
# 123

# Convert a matrix to a vector
m <- matrix(1:4, nrow = 2)
v <- as.vector(m)
print(v)  
# 1 3 2 4
```
