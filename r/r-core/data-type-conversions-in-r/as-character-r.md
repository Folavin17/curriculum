# as.character

---

## Content

The `as.character` converts an object to character (string) type.

```r
# Convert a number to a string
x <- 123
x <- as.character(x)
print(x)  
# "123"

# Convert a vector of numbers to a vector of strings
v <- c(1, 2, 3)
v <- as.character(v)
print(v)  
# "1" "2" "3"
```
