# as.numeric

---

## Content

The `as.numeric` function converts an object to numeric (integer or double) type.

Here's an example
```r
# Convert a string to a number
x <- "123"
x <- as.numeric(x)
print(x)  
# 123

# Convert a vector of strings to a vector of numbers
v <- c("1", "2", "3")
v <- as.numeric(v)
print(v)  
# 1 2 3
```
