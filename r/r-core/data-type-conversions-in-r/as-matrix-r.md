# as.matrix

---

## Content

The `as.matrix` converts an object to matrix type.

```r
# Convert a vector to a matrix
v <- c(1, 2, 3, 4)
m <- as.matrix(v)
print(m)  
#      [,1]
# [1,]    1
# [2,]    2
# [3,]    3
# [4,]    4

# Convert a list to a matrix
l <- list(c(1, 2), c(3, 4))
m <- as.matrix(l)
print(m)  
#      [,1] [,2]
# [1,]    1     2
# [2,]    3     4
```
