# as.integer

---

## Content

`as.integer`: Converts an object to integer type.

```r
# Convert a number to an integer
x <- 123.45
x <- as.integer(x)
print(x)  
# 123

# Convert a vector of numbers to a vector of integers
v <- c(1.5, 2.5, 3.5)
v <- as.integer(v)
print(v)  
# 1 2 3
```
